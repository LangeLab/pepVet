# Code of Conduct

## Our standard

pepVet aims to be a professional, respectful project for scientific software work. Contributors, users, and maintainers are expected to communicate directly, assume good faith, and critique ideas without attacking people.

Examples of expected behavior:

- Use respectful, technical, and constructive language.
- Ask questions when assumptions are unclear.
- Accept review feedback in good faith and respond with evidence.
- Keep scientific claims precise and proportionate to the evidence.

Examples of unacceptable behavior:

- Harassment, intimidation, or personal attacks.
- Discriminatory or exclusionary language.
- Sexualized language or unwanted attention.
- Deliberate misrepresentation of results, data, or contributor intent.
- Bad-faith review behavior, including hostile pile-ons or repeated personal escalation.

## Scope

This code of conduct applies in project issues, pull requests, discussions, documentation reviews, and other public spaces associated with the repository.

## Reporting

If you experience or witness unacceptable behavior, contact the maintainer at [eneskemalergin@gmail.com](mailto:eneskemalergin@gmail.com). Include links or screenshots when possible so the report can be evaluated quickly.

## Enforcement

The maintainer may remove comments, close discussions, reject contributions, or block participation when conduct violates this policy.
