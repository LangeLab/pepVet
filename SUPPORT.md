# Support

Use the channel that best matches the request.

## Documentation

- Start with the README and pkgdown site: [langelab.github.io/pepVet](https://langelab.github.io/pepVet/)
- Check the reference pages and articles before opening a support request.

## Questions and usage help

Open a GitHub issue if:

- the package behavior is unclear
- the docs are missing an example you needed
- you are unsure whether a workflow is in scope

Include the function call, package version, and a minimal reproducible example when possible.

## Bug reports

Use the bug report template and include:

- the input sequence or fixture used
- the expected behavior
- the observed behavior
- session information if the problem may be environment-specific

## Security reports

Follow [SECURITY.md](SECURITY.md) for vulnerabilities or sensitive disclosures.
